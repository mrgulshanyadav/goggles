# Brave Search Goggles — Misar AI Technology

Custom [Brave Search Goggles](https://search.brave.com/help/goggles) maintained by [Misar AI Technology](https://misar.io), [Assisters LLC](https://assisters.io), and [G1 Technologies](https://g1technologies.online).

Goggles are custom re-ranking filters for [Brave Search](https://search.brave.com) that boost quality, independent resources and downrank SEO spam — giving you a cleaner, more relevant search experience for your niche.

---

## Available Goggles

### 1. Master — Developer Tools & SaaS
**File:** [`master.goggle`](./master.goggle)

The broadest Goggle. Covers all Misar AI, Assisters, and G1 Technologies domains alongside a curated list of quality developer communities, AI platforms, mobile dev resources, and SaaS tools. Downranks known SEO spam and content farms.

**Best for:** Developers, SaaS founders, and AI builders who want cleaner general search results.

**Apply it:**
```
https://search.brave.com/goggles?goggles_id=https://raw.githubusercontent.com/mrgulshanyadav/goggles/main/master.goggle
```

## How to Use a Goggle

1. Copy any of the "Apply it" links above
2. Paste it into your browser while on [search.brave.com](https://search.brave.com)
3. Or go to the Brave Search **Goggles tab** and paste the raw GitHub URL

To submit a Goggle to the Brave Search directory:
1. Go to [search.brave.com/goggles/create](https://search.brave.com/goggles/create)
2. Paste the raw GitHub URL of the `.goggle` file
3. Click Submit
